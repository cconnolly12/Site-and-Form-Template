Use <br/> tag sparingly

The <p> tag contains a 15px top margin defined in the style sheet, making the <br/> tag redondant. Also the <br/> tag is font dependent and line height dependent. It is a wrong way to insert space in content or in a page.
the only proper use of the <br/> tag is to force a line break in text INSIDE a <p> or <div> container.

This line is in fact a new paragraph. You can observe the neat spacing without the use of <br/> tag

The <h3> tag has its margin set to zero since paragraph are spacing themselves away
